# CV Builder

Mobile-first resume builder. Edit on your phone with a bottom tab bar, preview live on desktop, print or save as PDF from the browser.

## Run locally

```bash
npm install
npm run dev
```

Open **http://localhost:3000** on your phone (same Wi‑Fi) or desktop.

> Production (Vercel) deploys from `main`. Merge the open PR to update the live site.

## Features

- Profile + work experience editor
- Auto-save in `localStorage`
- Mobile: Edit / Preview / PDF bottom navigation
- Desktop: editor + live preview side by side
- Print-friendly layout
